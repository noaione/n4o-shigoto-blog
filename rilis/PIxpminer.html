What is this: Penambang XP untuk web PerpusIndo menggunakan wget untuk prosesnya.

Kenapa buat ini: Kali aja ada fitur yang menggunakan xp nanti wkwk

Apa yang dibutuhkan: wget (sudah tersedia)

===== Cara menggunakan =====

1. Extract

2. klik penambang.bat

3. Masukan ID Berkas (ID berkas: http://www.perpusindo.info/berkas/*idberkas*), ex: http://www.perpusindo.info/berkas/selfjwzw

Dimana selfjwzw adalah id Berkas

4. Enter, dan tunggu

5. Kalo mau udahan tinggal pencet CTRL+C di keyboard, terus ketik 'y' (tanpa kutip)

Download: github
