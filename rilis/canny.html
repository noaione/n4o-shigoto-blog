Canny Edge Detector (using python 2.7)

Modules: Python 2.7

Rest of them included in zip files a.k.a modules

===== How to use =====

1. install all prerequisites with installdep.bat

2. run canny.bat with cmd like this > "cannny **PICTUREFILENAME**", example "canny asd.png" (without quote marks)

3. or if you prefer using python script > "canny.py -i picturefilename", example "canny.py -i asd.png" (without quote marks)

Download: github
